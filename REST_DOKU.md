#REST Dokumentation
##Abrufen von Fahrzeugtypen:
###alle Datensätze
```
rest/GET/TYP
```
###einelnen Datensatz
```
rest/GET/TYP/{ID}
```
###Bsp.: 
```
rest/GET/KFZ/32
```

##Abrufen von Fahrzeugdaten:
###alle Datensätze
```
rest/GET/KFZ
```
###einelnen Datensatz
```
rest/GET/KFZ/{KENNZEICHEN}
```
###Bsp.: 
```
rest/GET/KFZ/KA-MM218
```

##Abrufen von Mitarbeiterdaten:
```
rest/GET/MA
```
###einelnen Datensatz
```
rest/GET/MA/{PERSONALNUMMER}
```
###Bsp: 
```
rest/GET/MA/678492
```
